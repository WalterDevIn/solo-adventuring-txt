# Solo Adventuring TXT

Prototipo de interfaz para un RPG de texto de navegador.

## Ejecutar

No requiere instalación ni compilación. Abrir `index.html` directamente o iniciar el repositorio con Live Server.

## Estado actual

- Pantalla negra con estética retro neón monocromática.
- Aberración cromática roja y azul muy leve.
- Caja de entrada visible y centrada al iniciar.
- Caja de salida oculta hasta enviar el primer texto.
- Aparición animada de la salida sobre la entrada.
- Riel lateral reservado para futuros marcadores y excepciones.
- Diseño adaptable a pantallas pequeñas.

La salida actualmente repite el texto ingresado. Es un comportamiento provisional para diseñar el juego alrededor de la interfaz antes de implementar el motor narrativo.
